# AutoNews Gradio App

This is a simple Gradio app deployed on Hugging Face Spaces.

## How to Run

1. Clone the repository.
2. Install the dependencies:
    ```bash
    pip install -r requirements.txt
    ```
3. Run the app:
    ```bash
    python app.py
    ```

## Deployment

This app is deployed on Hugging Face Spaces.
